## 结构
* api - api相关
* dp - 设计模式
* ioex - io扩展
  * ioos - 系统文件
  * iopath - 路径
* object - 基础对象相关
* osex - 系统相关
* plugin - 插件(第三方)
  * db - 数据库相关
    * identity - 标识
    * mongodb - mongo相关
    * mysqldb - mysql相关
  * ginex - gin相关(http请求)
  * redisex - redis相关
  * thread - 锁
